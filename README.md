# 🕵️ 《Gleaner》— Gleaner 偵查系統

一個專為社群觀測與輿情分析設計的高質感、極簡化情報探勘終端。使用者只需輸入 YouTube 影片連結，系統將全自動、高速攻破留言區，精準攔截前排數據並即時解構觀眾的真實反應。

---

## ⚡ 核心功能
* **流體漸層開場動畫**：極簡的 5 秒初始化進度條，具備前衛的科技儀式感。
* **最高讚數留言 Top 5**：精準捕捉獲得最高迴響、被瘋狂點讚的神回覆。
* **進階文字探勘統計**：整合 `Jieba` 斷詞引擎並自訂贅字過濾，自動提煉出前 10 名核心關鍵字並圖像化呈現。
* **動態側邊欄設定**：內建留言數需求面板，可自由展開並調整要偵查的留言數量上限（100 - 1000）。
* **極客深色主題**：採用深灰藍色背景設計（#1a1c23），護眼、前衛且低調。
